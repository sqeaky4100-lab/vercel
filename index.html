▲ HTML drop folder ready

Any .html file committed to /html-drop in this repository will be deployed automatically by the Vercel project already connected to the repo.

How to use it

  1. Put your file in /html-drop.
  2. Commit and push the change to this repository.
  3. Wait for the Vercel deployment to finish.
  4. Open it at /files/your-file-name or /html-drop/your-file-name.html.
