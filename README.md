# vercel

Static HTML files for a Vercel-connected repository.

## HTML drop folder

Put any personal `.html` file into `/home/runner/work/vercel/vercel/html-drop` (the `html-drop` folder in this repository), then commit and push it. The connected Vercel project will deploy it automatically.

You can open uploaded files with either of these URL patterns:

- `/files/your-file-name`
- `/html-drop/your-file-name.html`

Example:

- `/files/Kayenta_Township_Audit_Knowledge_2022`
